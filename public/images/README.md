## Image Specifications:

- **Format**: JPEG or WebP for best performance
- **Hero Image**: 1920x1080px (16:9 aspect ratio)
- **Main Image**: 800x450px (16:9 aspect ratio)
- **Space Images**: 600x400px (3:2 aspect ratio)
- **Quality**: 80-85% compression for web optimization
